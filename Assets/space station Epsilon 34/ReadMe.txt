Space station "Epsilon 34" SCI-FI Modular environment pack

A customizable set of modular pieces that can be used to construct interior scenes in the sci-fi style. This pack consists of an extensive library of over 120 high quality prefabs (walls , floors, pipes, monitors, pre-built corridors, rooms, and more...). 


All textures provided are 2048x2048 size.
*Albedo (Diffuse)
*Metallic
*Normal 
*Emission (Some objects)


!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
WARNING

The doors will open only to the gameObject with the tag "player".
